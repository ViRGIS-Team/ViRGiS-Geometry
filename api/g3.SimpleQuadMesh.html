    Class SimpleQuadMesh

    SimpleTriangleMesh but for quads. Data packed into buffers, no dynamics. Supports Per-Vertex Normals, Colors, UV, and Per-Quad Facegroup.

    use static WriteOBJ() to save. No loading, for now.
